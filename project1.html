Anxiiety.com

Like a book, but on the internet!

All jokes aside, I've been planning to build a website to display my writing for years. So when I started building my own sites, making this one of the first felt like a no-brainer. My aims were simple: it should be fast, cheap to host, and easy for my grandma to use. The hardest (and coicidentally, the most enjoyable) part of this project was figuring out exactly how I wanted to implement it. I knew that I wanted it to be simple, but expressive. Using cards for the layout did exactly that, enabling me to easily communicate their purpose as navigation while using them for a light touch of style.
